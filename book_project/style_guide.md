# KDP Formatting Style Guide (2026 Standards)

## 1. Typography
- **Body Text**: Use standard serif fonts (Georgia, Palatino).
- **Font Size**: 11pt or 12pt for body; 18pt-24pt for Chapter Headings.
- **Line Spacing**: 1.15 to 1.3.

## 2. Layout
- **Margins**: 0.5" minimum.
- **Indents**: First-line indent of 0.3" for paragraphs (except the first paragraph of a chapter).
- **Alignment**: Justified or Left-aligned.

## 3. Structure
- **Chapter Starts**: Must use `Heading 1` tags.
- **Page Breaks**: Inserted automatically by KDP between Heading 1 sections.

## 4. Images
- **Resolution**: Minimum 300 DPI.
- **Alt-Text**: Mandatory for all images for accessibility.

## 5. AI Disclosure
- All content generated by AI must be disclosed as "AI-generated" during the KDP upload process.
